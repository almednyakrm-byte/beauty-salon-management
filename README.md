# نظام إدارة صالون تجميل (مواعيد، خدمات، دفع)
==========================

## ملخص المشروع
----------------

نظام إدارة صالون تجميل هو تطبيق مفتوح المصدر يهدف إلى تسهيل إدارة المواعيد، والخدمات، والعمليات المالية في صالون تجميل. يهدف المشروع إلى توفير حلًا متكاملًا لتحسين كفاءة العمل في الصالون وتحسين تجربة العملاء.

## هيكل المشروع
----------------

### مكونات المشروع

* `backend`: تطبيق Node.js يحتوي على لوجيكة الخدمات والعمليات المالية.
* `frontend`: تطبيق React يحتوي على واجهة المستخدم وتحكم المستخدم.
* `database`: قاعدة بيانات PostgreSQL تحتوي على جداول المواعيد والخدمات والعمليات المالية.

## تشغيل البيئة باستخدام docker-compose
----------------------------------------

### خطوات تشغيل البيئة

1. تأكد من وجود Docker وdocker-compose مثبتين على جهازك.
2. انتقل إلى مجلد المشروع.
3. تشغيل الأمر التالي لتشغيل البيئة:

bash
docker-compose up


### تفاصيل تشغيل البيئة

* يُنشئ docker-compose بيئة تشغيل تطبيق Node.js وReact وقاعدة بيانات PostgreSQL.
* يُنشئ docker-compose شبكة لتواصل بين التطبيقات.
* يُنشئ docker-compose ملفات تعريف الخادم لخدمات Node.js وReact.

## مكونات النظام
----------------

### جداول قاعدة البيانات

* `appointments`: جدول يحتوي على المواعيد.
* `services`: جدول يحتوي على الخدمات.
* `payments`: جدول يحتوي على العمليات المالية.

### أدوار النظام

* `admin`: دور يحتوي على صلاحيات إدارة النظام.
* `user`: دور يحتوي على صلاحيات استخدام النظام.
